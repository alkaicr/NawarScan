# NawarScan

**Professional Multi-threaded TCP Port Scanner**  
Developed by **Ahmed Ben Nawar**  

NawarScan is a professional tool designed for penetration testers, cybersecurity enthusiasts, and lab environments. It supports fast, multi-threaded scanning, banner grabbing, and flexible port ranges. Ideal for educational and authorized penetration testing.

---

## Features

- ✅ Multi-threaded TCP port scanning for high-speed performance  
- ✅ Custom port range scanning (any range between 1-65535)  
- ✅ Banner grabbing for service detection  
- ✅ Timeout control for faster scanning  
- ✅ Execution time measurement  
- ✅ Input validation to prevent invalid ports
